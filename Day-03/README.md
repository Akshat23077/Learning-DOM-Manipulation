# 📅 Day 03 - DOM Manipulation Project

## 🚀 Project Title

**Dynamic User Card Generator (Form Handling)**

---

## 📌 Description

This project demonstrates how to handle form inputs using **JavaScript DOM Manipulation**.
Users can enter their details in a form, and the data can be processed dynamically (via JavaScript) to create UI elements like user cards.

---

## 🎯 Features

* 📝 Form with multiple input fields (Name, Age, Email, Profile Pic)
* ⚡ Prevent default form submission
* 🎨 Styled UI with card layout
* 🧠 DOM manipulation using JavaScript
* 📦 Dynamic content rendering (via JS)

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

---

## 📂 Folder Structure

```
Day-03/
│── index.html
│── script.js
```

---

## 📸 How It Works

1. User fills the form
2. JavaScript captures input values
3. Prevents page reload
4. Creates a new user card dynamically

---

## 📚 Concepts Covered

* DOM Selection (`querySelector`)
* Event Handling (`addEventListener`)
* Preventing default behavior (`event.preventDefault()`)
* Dynamic element creation
* Updating UI using JavaScript

---
