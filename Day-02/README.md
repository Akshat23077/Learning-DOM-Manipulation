# Day 02 - Custom File Upload Button 📁

## 📌 What I built

A custom file upload button that replaces the default file input UI and displays the selected file name.

## 🧠 What I learned

* Handling file inputs in JavaScript
* Using `click()` to trigger hidden input
* Accessing selected files via `event.target.files`
* Updating UI dynamically

## ⚡ Features

* Custom styled upload button
* Hidden file input
* Displays selected file name

## 🚀 Future Improvements

* Show file size
* Validate file type
* Add upload preview
